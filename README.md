# tlon

Terminal based quantum circuit simulator.

### Roadmap
- Develop ncurses based modular system for windows/panels
- Write linear algebra functions, tensor products, matrix multiplications, for arbitrary sized matrices (math.cpp)
- Figure out UI design
- Figure out a way to serialize circuit, and draw them in terminal (this will probably be a pain)
- Figure out base set of quantum gates
- Figure out way to store and load circuits
- Export/import circuits to/from qiskit
- Import matrix as custom gate
- Save subcircuit as custom gate
- Run circuit with given inputs and get outputs
